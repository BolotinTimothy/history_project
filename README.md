# Telegram-бот с интерактивными историческими сюжетами

Приложение представляет собой Telegram-бота на Python c SQLite-базой данных. Пользователь выбирает одну из историй о событиях Великой Отечественной войны, проходит её по этапам и на каждом шаге видит, какой вариант был исторически верным и почему.

Сейчас в проекте добавлены две демонстрационные истории-заглушки, чтобы можно было проверить механику. Позже их заменим на документально подтверждённые сюжеты.

## Что умеет бот

- показывает список доступных историй;
- умеет выбирать случайную историю;
- хранит истории, этапы и варианты ответов в базе данных;
- хранит для каждого варианта отдельное последствие выбора, которое показывается пользователю;
- хранит текущее состояние чата, чтобы понимать, какой этап проходит пользователь;
- после любого выбора объясняет правильный вариант и переводит пользователя к следующему этапу;

## Структура проекта

```text
app/
  bot.py         # логика Telegram-бота
  config.py      # загрузка настроек
  database.py    # схема БД, сидирование, работа с сессиями
stories/
  *.json         # истории для загрузки в БД
main.py          # точка входа
```
